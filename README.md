# Classical Cryptography Toolkit

A Python project demonstrating the implementation of classical cryptographic algorithms. This toolkit includes encryption and decryption programs for some of the most widely studied classical ciphers and is intended for educational purposes.

## Features

* Caesar Cipher
* Monoalphabetic Cipher
* Vigenère (Polyalphabetic) Cipher
* Encryption and decryption functionality
* Simple command-line interface

## Technologies Used

* Python 3

## Purpose

This project was developed to understand the fundamentals of classical cryptography, encryption techniques, and secure communication concepts. It serves as a learning resource for students studying cryptography and cybersecurity.

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project folder:

   ```bash
   cd Classical-Cryptography-Toolkit
   ```

3. Run the desired Python file:

   ```bash
   python caesar_cipher.py
   ```

## Learning Outcomes

* Understanding substitution and polyalphabetic ciphers
* Implementing encryption and decryption algorithms in Python
* Strengthening Python programming and problem-solving skills
* Exploring foundational cryptography concepts

## Disclaimer

This project implements **classical cryptographic algorithms for educational purposes only**. These ciphers are **not secure for protecting real-world sensitive information** and should not be used in modern security applications.
